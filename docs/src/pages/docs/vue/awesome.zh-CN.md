## 官方生态

| 名称 | 详情 |
| ---- | ---- |
| [vue-components](https://github.com/antdv-next/vue-components) | 底层 `@v-c/*` 的无头样式组件 |

| [@antdv-next/auto-import-resolver](https://github.com/antdv-next/auto-import-resolver) | 按需引入配置文件 |
| [@antdv-next/icons](https://github.com/antdv-next/icons) | SVG 格式的图标 |
| [@antdv-next/nuxt](https://github.com/antdv-next/nuxt) | Nuxt 模块 |
| [admin](https://github.com/antdv-next/admin) | 后台管理系统模板 |
| [antdv-style](https://github.com/antdv-next/antdv-style) | CSS-in-JS 解决方案，基于 `Emotion` |
| [antdv-next-playground](https://github.com/antdv-next/antdv-next-playground) | 在线演练场，基于 `@vue/repl` |
| [css-plugin](https://github.com/antdv-next/css-plugin) | `@antdv-next/unocss`、`@antdv-next/tailwind` 插件 |

| [vite-plugin-dayjs](https://github.com/antdv-next/vite-plugin-dayjs) | 将 Day.js 的 CommonJS 导入转换为 ESM 格式 |
